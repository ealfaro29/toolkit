# Theme Asset Structure

This folder contains visual theme assets used by the hub and tools.

## Folder Convention

- `Theme/comicbook/`
- `Theme/glassui/`
- `Theme/minimal/`
- `Theme/emoji/`

## Theme Groups

- `Visual`
  - `comicbook`
  - `glassui`
- `Basic`
  - `minimal`
  - `emoji`

## File Naming Convention

- Tool previews: `<toolname>_<themename>.png`
  - Example: `mockups_comicbook.png`
  - Example: `svgrecolor_glassui.png`
- Hub background pattern: `pattern_<themename>.jpg`
  - Example: `pattern_comicbook.jpg`
  - Example: `pattern_glassui.jpg`

## Asset-Less Themes

- `minimal` is a `Basic` theme and intentionally does not use splash or card artwork.
- It relies on monochrome surfaces, typography, and restrained contrast instead of preview images.
- `emoji` is a `Basic` theme that replaces artwork with oversized emoji patterns tailored to each tool.
