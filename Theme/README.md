# Theme Asset Structure

This folder contains visual theme assets used by the hub and tools.

## Folder Convention

- `Theme/comicbook/`
- `Theme/glassui/`

## File Naming Convention

- Tool previews: `<toolname>_<themename>.png`
  - Example: `mockups_comicbook.png`
  - Example: `svgrecolor_glassui.png`
- Hub background pattern: `pattern_<themename>.jpg`
  - Example: `pattern_comicbook.jpg`
  - Example: `pattern_glassui.jpg`
