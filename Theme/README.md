# Theme Asset Structure

This folder contains visual theme assets used by the hub and tools.

## Folder Convention

- `Theme/comicbook/`
- `Theme/glassui/`

## File Naming Convention

- Tool previews: `<toolname>_<themename>_<mode>.png`
  - Example: `mockups_comicbook_light.png`
  - Example: `svgrecolor_glassui_dark.png`
- Hub background pattern: `pattern_<themename>_<mode>.jpg`
  - Example: `pattern_comicbook_light.jpg`
  - Example: `pattern_glassui_dark.jpg`

## Modes

- `light`
- `dark`

If a new theme is added, provide both mode variants for all tool preview images and pattern assets.
