SVG Recolor Info

How to Use

1. Upload SVG: Drop an SVG file or click to browse.

2. Review Colors: All colors are automatically detected and grouped by hue.

3. Change Colors: Click any color swatch to pick a new color.

4. Export: Download your recolored SVG file.

Features

Smart Grouping: Colors are organized by hue (reds, blues, greens, etc.)

Real-time Preview: See changes instantly as you pick new colors.

PowerPoint Ready: Exported SVGs work perfectly in presentations.

100% Private: All processing happens in your browser.
