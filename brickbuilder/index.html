How to Use

  • Add Bricks: Click on any brick in the bottom Dock to add it.
  • Move: Drag bricks to position them. They snap to the grid.
  • Customize: Select a brick to change size and color in the panel.
  • Custom Colors: Use the Custom color picker for any RGB color.
  • Shortcuts:
    • R: Rotate
    • Del: Delete
    • D: Duplicate
    • H: Hand / Pan (Toggle)
    • V: Select Tool
    • Space: Hold to Pan
    • Ctrl+Z/Y: Undo/Redo
    • Ctrl+G: Group / Ungroup

Features v2.3.0

  • Smart Bricks: Resizable bricks, slopes, and arches.
  • Auto-Save: Work saved every 30s automatically.
  • Export: Save as SVG format.
  • Custom Colors: Any color via color picker.
  • Selection Info: Live brick count and type display.
  • Grouping: Group multiple bricks together.
  • Alignment: Align and distribute tools.
  • Themes: Light and Dark mode support.
